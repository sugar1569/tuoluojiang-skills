> 版本：1.0.0
> 作者：陀螺匠团队

## 7.1 路由概述

### 7.1.1 路由定义方式

本系统基于 Laravel 9.x 框架，采用以下几种方式定义路由：

1. **路由属性（Route Attributes）**：推荐使用，在控制器方法上直接定义路由
2. **传统路由文件**：在 `routes` 目录下定义路由
3. **服务提供者**：在 `RouteServiceProvider.php` 中定义全局路由

### 7.1.2 路由文件结构

```
routes/
├── README.md          # 路由目录说明
├── web.php            # Web路由文件（当前为空）
├── console.php        # 控制台路由文件（当前为空）
├── channels.php       # 广播通道路由文件（当前为空）
└── websocket.php      # WebSocket路由文件（当前为空）
```

## 7.2 核心路由配置

### 7.2.1 RouteServiceProvider 配置

系统在 `App\Providers\RouteServiceProvider.php` 中配置了全局路由：

```php
// 问卷调查中转页面
Route::get('q/{unique}', [CommonController::class, 'questionnaire']);

// 获取版本信息
Route::get('api/version', [CommonController::class, 'getVersion']);

// 安装程序路由
Route::get('install/index/{step}', [Install::class, 'index']);
Route::post('install/index/{step}', [Install::class, 'index']);

// Web路由组
Route::middleware('web')->namespace($this->namespace)->group(base_path('routes/web.php'));
```

### 7.2.2 路由兜底处理

系统配置了路由兜底（fallback）处理，用于处理未匹配的路由：

```php
Route::fallback(function () {
    $appRequest = request()->getPathInfo();
    if ($appRequest === null) {
        $appName = config('app.ent.name');
    } else {
        $appRequest = str_replace('//', '/', $appRequest);
        $appName    = explode('/', $appRequest)[1] ?? '';
    }
    // 检测是否已安装系统
    if (! file_exists(public_path('install/install.lock'))) {
        return redirect('/install/index/1');
    }
    if ($appName === 'api') {
        return app('json')->make(400, 'common.route.miss');
    }
    if (! sys_config('ent_site_open', 1)) {
        return Response::make('网站已关闭');
    }
    if ($appName === 'chat') {
        return View::file(public_path('chat') . DIRECTORY_SEPARATOR . 'index.html');
    }
    $userAgent = request()->header('User-Agent', 'admin');
    if (str_contains($userAgent, 'Mobi') || (str_contains($userAgent, 'wxwork') && $appName === 'work')) {
        // 手机端访问
        return View::file(public_path('work') . DIRECTORY_SEPARATOR . 'index.html');
    }
    // 非手机端访问
    return View::file(public_path('admin') . DIRECTORY_SEPARATOR . 'index.html');
});
```

## 7.3 路由类型

### 7.3.1 系统路由类型

| 路由类型 | 描述 | 示例 |
|---------|------|------|
| Web路由 | 网页访问路由 | `/admin`、`/chat` |
| API路由 | 接口访问路由 | `/api/version` |
| 安装路由 | 系统安装路由 | `/install/index/{step}` |
| 问卷调查路由 | 问卷调查中转路由 | `/q/{unique}` |
| WebSocket路由 | WebSocket连接路由 | 配置在 `config/laravels.php` |

### 7.3.2 菜单与路由关系

系统菜单与路由的关系通过数据库表 `system_menus` 维护：

| 字段名 | 描述 | 示例 |
|-------|------|------|
| `menu_path` | 前端路由路径 | `/admin/user/index` |
| `uni_path` | 移动端路径 | `/pages/user/index` |
| `component` | 前端组件路径 | `user/index` |
| `paths` | 路径 | `user/index` |
| `uniqued` | 菜单唯一标识 | `menus123456` |
| `type` | 类型：M、菜单；B、按钮；A、接口 | `M` |

## 7.4 路由工具类

### 7.4.1 MenusRouteTrait

系统提供了 `MenusRouteTrait` 用于处理菜单和路由的关系：

```php
use crmeb\traits\service\MenusRouteTraits;

class MenuService
{
    use MenusRouteTraits;
    
    // 获取全部路由
    public function getAllRoutes()
    {
        return $this->getRoutes(function ($route) {
            return $route;
        });
    }
    
    // 过滤路由
    public function getFilteredRoutes($name, $path, $method)
    {
        $this->setOption('name', $name);
        $this->setOption('path', $path);
        $this->setOption('method', $method);
        
        return $this->getRoutes(function ($route) {
            return $this->filterRoute($route);
        });
    }
}
```

### 7.4.2 路由辅助函数

系统提供了获取路由模块名的辅助函数：

```php
/**
 * 获取配置中的路由模块名
 *
 * @param string $value 附加路径
 * @param int $type 类型：0=admin，1=ent
 * @return string
 */
function get_roule_mobu(string $value = '', int $type = 0)
{
    if ($type) {
        $typeName = 'ent';
    } else {
        $typeName = 'admin';
    }
    return config('app.' . $typeName . '.path', '/admin') . $value;
}
```

## 7.5 路由使用指南

### 7.5.1 使用路由属性定义路由

**示例：**

```php
namespace App\Http\Controller\AdminApi;

use App\Http\Controller\BaseController;
use Illuminate\Routing\Route;

class UserController extends BaseController
{
    /**
     * @param Route $route
     */
    #[\App\Annotation\Route(path: '/adminapi/user/list', methods: ['GET'])]
    public function list()
    {
        // 方法实现
    }
    
    /**
     * @param Route $route
     */
    #[\App\Annotation\Route(path: '/adminapi/user/create', methods: ['POST'])]
    public function create()
    {
        // 方法实现
    }
}
```

### 7.5.2 传统路由文件定义

**示例：**

在 `routes/web.php` 中定义：

```php
Route::middleware('web')->group(function () {
    // 后台路由
    Route::prefix('admin')->group(function () {
        Route::get('/', [AdminController::class, 'index']);
        Route::resource('user', UserController::class);
    });
    
    // 前端路由
    Route::prefix('api')->group(function () {
        Route::get('version', [CommonController::class, 'getVersion']);
    });
});
```

### 7.5.3 路由参数传递

**示例：**

```php
// 路由定义
# [\App\Annotation\Route(path: '/adminapi/user/{id}', methods: ['GET'])]
public function show(int $id)
{
    // 使用 $id 参数
}

// 路由调用
// GET /adminapi/user/1
```

### 7.5.4 路由中间件

**示例：**

```php
// 单个中间件
# [\App\Annotation\Route(path: '/adminapi/user/list', methods: ['GET'], middleware: ['auth:admin'])]
```

## 7.8 路由最佳实践

### 7.8.1 命名规范

1. **路由路径**：使用小写字母，单词间用连字符 `-` 分隔
   ```php
   // 推荐
   #[\App\Annotation\Route(path: '/adminapi/user/list', methods: ['GET'])]
   ```

2. **路由命名**：使用 snake_case 命名路由
   ```php
   // 推荐
   Route::get('/user/list', [UserController::class, 'list'])->name('user.list');
   ```

### 7.8.2 性能优化

1. **使用路由缓存**：生产环境启用路由缓存
2. **合理分组路由**：使用路由组减少重复代码
3. **避免过度使用中间件**：只在必要的路由上使用中间件
4. **使用路由模型绑定**：减少数据库查询

### 7.8.3 安全建议

1. **验证所有输入**：使用请求验证类验证路由参数
2. **使用 HTTPS**：生产环境强制使用 HTTPS
3. **限制路由访问**：使用中间件限制敏感路由的访问
4. **防止 CSRF 攻击**：对 POST、PUT、DELETE 请求使用 CSRF 令牌
