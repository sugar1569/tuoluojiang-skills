# 陀螺匠 API 常用命令

> 版本：1.0.0
> 作者：陀螺匠团队
> 更新日期：2026-03-24

## 6.1 开发环境命令

### 6.1.1 启动开发服务器

```bash
# 使用 PHP 内置服务器
php artisan serve

# 指定端口和主机
php artisan serve --host=0.0.0.0 --port=8000

# 使用 LaravelS (Swoole)
php bin/laravels start
php bin/laravels start -d  # 守护进程模式
```

### 6.1.2 环境配置

```bash
# 生成应用密钥
php artisan key:generate

# 复制环境变量文件
cp .env.example .env

# 清除配置缓存
php artisan config:clear

# 缓存配置
php artisan config:cache

# 查看当前环境配置
php artisan env
```

## 6.2 数据库命令

### 6.2.1 数据库迁移

```bash
# 运行所有迁移
php artisan migrate

# 回滚最后一次迁移
php artisan migrate:rollback

# 回滚所有迁移并重新运行
php artisan migrate:fresh

# 回滚所有迁移
php artisan migrate:reset

# 查看迁移状态
php artisan migrate:status

# 创建迁移文件
php artisan make:migration create_users_table

# 创建表迁移示例
php artisan make:migration create_new_module_table --create=new_module
```

### 6.2.2 数据库种子

```bash
# 运行所有种子
php artisan db:seed

# 运行指定种子
php artisan db:seed --class=UserSeeder

# 迁移并种子
php artisan migrate:fresh --seed

# 创建种子文件
php artisan make:seeder UserSeeder
```

### 6.2.3 数据库工厂

```bash
# 创建工厂文件
php artisan make:factory UserFactory

# 使用 tinker 调试
php artisan tinker

# 在 tinker 中创建测试数据
>>> factory(App\Models\User::class, 10)->create();
```

## 6.3 路由命令

### 6.3.1 路由查看

```bash
# 查看所有路由
php artisan route:list

# 查看路由详细信息
php artisan route:list -v

# 过滤路由
php artisan route:list --path=api
php artisan route:list --method=GET

# 查看路由缓存
php artisan route:cache

# 清除路由缓存
php artisan route:clear
```

### 6.3.2 路由创建

```bash
# 创建控制器
php artisan make:controller UserController

# 创建资源控制器
php artisan make:controller UserController --resource

# 创建 API 控制器
php artisan make:controller UserController --api

# 创建控制器并指定模型
php artisan make:controller UserController --model=User
```

### 6.4.1 模型创建

### 6.4.2 模型命令

```bash
php artisan make:observer UserObserver
```


## 6.6 队列命令

### 6.6.1 队列处理

```bash
# 启动队列处理器
php artisan queue:work

# 使用 Redis 队列
php artisan queue:work redis

# 指定队列
php artisan queue:work redis --queue=emails

# 守护进程模式
php artisan queue:work --daemon

# 限制任务执行时间
php artisan queue:work --timeout=60

# 限制重试次数
php artisan queue:work --tries=3
```

### 6.6.2 队列管理

```bash
# 监听队列
php artisan queue:listen

# 重启队列处理器
php artisan queue:restart

# 清空队列
php artisan queue:clear

# 查看失败的任务
php artisan queue:failed

# 重试失败的任务
php artisan queue:retry all
php artisan queue:retry 5  # 重试指定 ID

# 清空失败的任务
php artisan queue:flush

# 创建任务类
php artisan make:job SendEmail

# 创建监听器
php artisan make:listener SendEmailListener
```

## 6.7 缓存命令

### 6.7.1 缓存操作

```bash
# 清除所有缓存
php artisan cache:clear

# 清除配置缓存
php artisan config:clear

# 清除路由缓存
php artisan route:clear

# 清除视图缓存
php artisan view:clear

# 缓存配置
php artisan config:cache

# 缓存路由
php artisan route:cache

# 缓存视图
php artisan view:cache

# 创建缓存表（用于数据库缓存驱动）
php artisan cache:table
```

### 6.7.2 缓存查看

```bash
# 查看缓存（需要安装 Laravel Tinker）
php artisan tinker
>>> Cache::get('key')

# 设置缓存
>>> Cache::put('key', 'value', $seconds)
```

## 6.9 测试命令

### 6.9.1 PHPUnit 测试

```bash

# 运行特定测试

php vendor/bin/phpunit --filter=testUserCanLogin

php vendor/bin/phpunit --coverage-html coverage
```

### 6.9.2 创建测试

```bash
# 创建测试类
php artisan make:test UserTest

# 创建单元测试
php artisan make:test UserTest --unit

# 创建 Feature 测试
```


## 6.12 Docker 和 Sail 命令

### 6.12.1 Laravel Sail

```bash
# 启动容器
./vendor/bin/sail up

# 后台运行
./vendor/bin/sail up -d

# 停止容器
./vendor/bin/sail down

# 执行命令
./vendor/bin/sail artisan migrate
./vendor/bin/sail artisan db:seed
./vendor/bin/sail php --version

# 查看日志
./vendor/bin/sail logs
./vendor/bin/sail logs -f
```

### 6.12.2 Docker Compose

```bash
# 启动服务
docker-compose up -d

# 停止服务
docker-compose down

# 查看日志
docker-compose logs -f

# 重建容器
docker-compose up -d --force-recreate
```

## 6.14 Git 命令

### 6.14.1 常用 Git 命令

```bash
# 克隆仓库
git clone <repository-url>

# 创建功能分支

git add .
```
