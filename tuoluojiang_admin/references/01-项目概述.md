# 项目概述

## 1. 项目简介

陀螺匠管理后台是一个基于 Vue 2.6.10 和 Element UI 2.15.14 开发的企业级管理系统前端模板。该项目旨在为企业提供一个功能完整、界面美观、易于扩展的管理后台解决方案。

### 1.1 项目特点

- **完整的技术栈**：Vue 2.6.10 + Vue Router 3.0.2 + Vuex 3.1.0 + Element UI 2.15.14
- **丰富的组件库**：内置多种业务组件和通用组件
- **集成多种功能**：富文本编辑器、Excel 表格处理、思维导图、文件预览等
- **响应式设计**：支持多种设备屏幕
- **国际化支持**：内置多语言切换功能
- **权限管理**：细粒度的权限控制
- **开发工具**：提供组件创建、页面创建等脚本工具

### 1.2 适用场景

- 企业管理系统
- OA 办公系统
- 后台管理平台
- 内容管理系统
- 数据管理系统

## 3. 开发环境

### 3.1 环境要求

- Node.js >= 8.9
- npm >= 3.0.0
- 现代浏览器（Chrome、Firefox、Safari、Edge）

### 3.2 安装依赖

```bash
# 安装依赖
npm install

# 使用 npm 镜像（推荐）
npm install --registry=https://registry.npmmirror.com
```

### 3.3 启动开发服务器

```bash
# 启动开发服务器
npm run dev

# 访问地址
# http://localhost:8080
```
