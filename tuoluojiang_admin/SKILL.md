# 陀螺匠管理后台技能文档

## 1. 技能概述

**技能名称**: tuoluojiang_admin
**技能版本**: 1.0.0
**技能描述**: 陀螺匠管理后台前端开发技能，提供完整的 Vue 管理后台模板和开发工具。
**适用场景**: 企业管理系统、OA 系统、后台管理平台等前端开发。
**触发条件**:
- 当用户询问关于 Vue 管理后台开发的问题时
- 当用户需要创建 Vue 组件或页面时
- 当用户询问关于项目结构、开发规范、API 接口等问题时
- 当用户需要使用开发工具或脚本时
- 当用户询问关于构建部署的问题时
- 当用户打开或编辑 `template` 目录下的文件时
- 当用户打开或编辑 JavaScript 文件（.js）时
- 当用户打开或编辑 Vue 组件文件（.vue）时
- 当用户打开或编辑 SCSS 样式文件（.scss）时

**触发关键词**:
- Vue 管理后台
- 组件开发
- 页面创建
- 项目结构
- 开发规范
- API 接口
- 构建部署
- 开发工具
- 脚本工具
- template 目录
- JavaScript 文件
- Vue 组件
- SCSS 文件
- .js 文件
- .vue 文件
- .scss 文件

**技能调用示例**:

1. **创建组件**:
   ```bash
   # 使用脚本创建组件
   node .trae/skills/tuoluojiang_admin/scripts/create-component.js --name MyComponent
   ```

2. **创建页面**:
   ```bash
   # 使用脚本创建页面
   node .trae/skills/tuoluojiang_admin/scripts/create-page.js --name MyPage
   ```

3. **代码检查**:
   ```bash
   # 运行代码检查
   node .trae/skills/tuoluojiang_admin/scripts/lint-check.js
   ```

4. **查询项目信息**:
   - "陀螺匠管理后台的技术栈是什么？"
   - "如何创建一个新的 Vue 组件？"
   - "项目的目录结构是怎样的？"

5. **文件操作触发**:
   - 打开 `template/src/components/common/echarts.vue` 文件
   - 编辑 `template/src/api/user.js` 文件
   - 打开 `template/src/assets/css/style.scss` 文件

**使用流程**:

1. **了解项目**：通过查看项目概述和目录结构，了解项目的基本情况
2. **准备环境**：按照快速开始指南安装依赖并启动开发服务器
3. **使用开发工具**：使用脚本工具创建组件、页面或检查代码
4. **参考文档**：查阅参考文档获取详细的开发指南
5. **构建部署**：按照部署指南构建和部署项目
